Shuttle.Core

The Shuttle.Core set of components consists of various assemblies that are used to facilitate the development of .Net software.

These components are used extensively in other repositories such as the Shuttle.Esb and Shuttle.Recall projects.

Now supports .Net 4.6+ and .Net Core 2.0+ (from v10.0.0)
